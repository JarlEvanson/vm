# REVM

## What is this about?

This is the main repository of the Reverse Engineering Virtual Machine (REVM)
project, which aims to provide a platform for hardware probing, remote
debugging, and black-box reverse engineering.

The project utilizes a two-stage protocol to load the project.

## System Support

### Stub Architecture and Boot Protocol Support

- Limine (`aarch64` and `x86_64`)
- Linux (`aarch64` and `x86_64`)
- UEFI (`aarch64` and `x86_64`)

### REVM Support

#### Architectures

- `aarch64`
- `i686`
- `x86_64`

#### Execution Support

- Limine (`aarch64` and `x86_64`)
- Linux (`aarch64` and `x86_64`)
- UEFI (`aarch64` and `x86_64`)
